El desarrollo de esta interfaz web que permite a los usuarios compartir y visualizar datos geográficos relacionados con eventos adversos recurrentes a través de Internet. Este logro se ha alcanzado mediante un proceso integral que incluye la recopilación de datos geográficos, la creación de un mapa web interactivo en la nube, la configuración de un servidor web local y el diseño de una interfaz web intuitiva. La integración de un mapa web en la interfaz, ya sea mediante la incrustación de un iframe o la inclusión de un Web Map Service (WMS) de la OGC, proporciona a los usuarios la capacidad de acceder y utilizar estos datos no solo en un navegador web, sino también en otro software de Sistemas de Información Geográfica (SIG). Esta interfaz web no solo cumple su objetivo específico, sino que también promueve la colaboración, la visualización y la aplicación de datos geográficos, contribuyendo así al conocimiento científico y a la toma de decisiones informadas relacionadas con eventos adversos recurrentes.
